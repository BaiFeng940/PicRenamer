# 📱 打包 APK 说明

## 快速打包步骤

### 1. 下载 Android Studio
- 官网：https://developer.android.com/studio
- 或国内镜像：https://developer.android.google.cn/studio

### 2. 打开项目
1. 启动 Android Studio
2. 点击 **Open** 或 **File → Open**
3. 选择 `BatchImageRenamer` 文件夹

### 3. 等待 Gradle 同步
- 首次打开会自动下载依赖
- 需要网络连接
- 大约需要 5-10 分钟

### 4. 编译 APK
1. 点击菜单 **Build**
2. 选择 **Build Bundle(s) / APK(s)**
3. 选择 **Build APK(s)**
4. 等待编译完成

### 5. 获取 APK
编译完成后会弹出提示，点击 **locate** 可以直接打开文件夹：
```
app/build/outputs/apk/debug/app-debug.apk
```

---

## 📋 系统要求

- **操作系统**：Windows 10/11, macOS 10.14+, Linux
- **内存**：至少 8GB RAM（推荐 16GB）
- **硬盘**：至少 10GB 可用空间
- **网络**：需要下载 SDK 和依赖

---

## 🔧 如果遇到问题

### 1. Gradle 同步失败
- 检查网络连接
- 尝试使用国内镜像源

### 2. SDK 缺失
- Android Studio 会自动提示安装
- 按照提示安装 SDK 34

### 3. 编译错误
- 确保 JDK 版本是 17+
- 清理项目：**Build → Clean Project**
- 重新构建：**Build → Rebuild Project**

---

## 📦 APK 安装

生成的 APK 可以直接安装到 Android 手机：
1. 将 APK 传到手机
2. 在手机上打开 APK 文件
3. 允许安装未知来源应用
4. 完成安装

---

## ⚠️ 注意事项

- 首次编译需要下载大量依赖，请耐心等待
- 确保有足够的磁盘空间
- 如果编译太慢，可以关闭其他程序
